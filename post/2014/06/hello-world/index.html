Hello, World!

N种语言的Hello, World!

  • Shell

    #!/bin/sh
echo "Hello, World!"

  • Fortran

    program main
    implicit none
    print *,"Hello, World!"
end program main

  • C

    #include <stdio.h>
int main()
{
    printf("Hello, World");
    return 0;
}

  • Python

        print("Hello, World!")

  • JavaScript

    console.log("Hello, World!");

  • PHP

    echo 'Hello, World!';

  • Go

    package main
import "fmt"
func main(){
    fmt.Print("Hello, World!");
}
